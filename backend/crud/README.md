# PHP CRUD (XAMPP)

## Requirements
- XAMPP (Apache + MySQL)

## Setup
1) Copy folder to: xampp/htdocs/crud
2) Start Apache & MySQL
3) Create DB in phpMyAdmin
4) Import: database/schema.sql (or database/database.sql) 
5) Copy config/db.example.php -> config/db.php and set DB name
6) Open: http://localhost/crud
